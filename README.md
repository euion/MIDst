## 1.3 프로젝트 기획 과정

---

### 1.3.1 기획 배경 📈

- 미드나 영화로 영어를 공부하는 학습자의 분포가 많은 것에 비하여 이를 뒷받침하는 용이한 서비스는 시중에 많지 않아 스스로 대본을 찾아서 공부하거나 자막을 보는 것으로 학습을 만족하는 학습자가 대부분입니다.

   <aside>

  **Midst**

  - 미드나 영화로 영어 학습 효과를 기대하는 사용자에게 단지 시청하는 것으로 끝나지 않도록 ai를 통한 정확한 표현 검색, 다이어리 작성과 문법 교정 등의 기능을 제공하여 학습 효과를 극대화시키고자 합니다.

   </aside>

  </br> </br>

   <aside>

  **영어교육 시장**

  - 세계의 디지털 영어학습 시장규모는 연평균 17%의 성장률을 기록하며 오는 2025년 123억8000만달러(약 16조3000억원) 규모로 커질 것으로 전망되고 있습니다.

  - 국내 영어학원은 3만8000개로 관련 콘텐츠 시장 규모는 1200억원 정도로 파악되고 있으며 영어학습에 대한 관심은 나날이 높아져 가고 있습니다.

  > 참고 자료 : 출처 : 더스탁(The Stock)(http://www.the-stock.kr)

   </aside>

### 1.3.2 문제 정의와 해결 방법 제안

<aside>

**문제 정의와 해결방법**

</aside>

- 미국 드라마나 영국 드라마를 활용하여 공부하는 이른바, 미드 공부법이 인기를 끌고 있습니다. 생활 속 다양한 영어표현을 재미있게 익히고자 하는 것에 초점을 둔 이른바 ‘미드공부법’은 많은 사람들에게 인기를 끌고 있지만 정작 초심자에게는 어떻게 미드를 활용하여 공부를 시작해야할지 모르는 경우가 대다수입니다.
- 미국 드라마의 스크립트를 활용하여 표현을 저장하고 나중에 학습자료로 활용하려고 할 때 번거로운 경우가 많습니다.
- 영어 작문에 어려움을 느껴 영어 일기를 작성하는 사람들이 많지만 문법 교정 및 작문교정을 하고 싶을 경우 어려움을 겪는 경우가 많습니다.

<aside>

**해결 방법**

</aside>

- 영어 일기 작성 도움 , 작성한 일기의 내용에 대한 감정분석,
- 특정 단어 및 문장으로 영어 대사 검색

### 1.3.3 타겟

<aside>

**영어 공부를 원하는 사람들**

**`정의`**

- 미드를 좋아하거나, 영어 일기 작성을 목표로 하는 영어 학습자
- 미국드라마나 영국 드라마를 활용하여 영어공부를 하고 싶은 이용자.
- 미드 시리즈의 스크립트를 통한 영어 표현을 바로 저장하고 싶은 이용자.
- 영어 일기 작성을 통해 문법 교정과 다양한 감정분석을 해보고 싶은 이용자.

</aside>

# 2. 서비스 기능 ⚙️

---

1. `영어 일기`
   - 작성된 영어 일기 번역 및 교정.
   - 작성된 영어 일기의 감정 분석
2. `커뮤니티`
   - 회원끼리 정보를 공유할 수 있는 커뮤니티 게시판을 제공.
   - 댓글 , 좋아요 기능
3. `회원가입 및 로그인`
   - Midst의 자체 회원가입 및 로그인 기능
   - 구글, 네이버의 소셜 로그인 기능
4. `검색 기능`
   - 사용자가 궁금한 영어 대사를 단어나 문장으로 검색
   - 유사도 측정하여 가장 비슷한 단어나 문장 출력
   - 출력된 결과를 필터 및 저장
5. `마이페이지`
   - 프로필 닉네임 수정 및 회원 탈퇴 기능
   - 저장된 대사나 컨텐츠 조회, 삭제 가능

# 3. 프로젝트 기술 스택

---

`프론트엔드`

> TypeScript, React, Recoil, Antd, CSS, HTML5

`사용 이유`

- 프론트엔드 개발을 맡아 진행하였기 때문에 프론트엔드 항목에 한하여 사용목적을 작성하였습니다.
  | 이름 | 역할 |
  | ------ | -------------------- |
  | TypeScript | 코드에 목적을 명시하고 목적에 맞지않는 타입에 대한 오류를 사전에 방지할 수 있기 때문에 사용하게 되었습니다. |
  | React | TypeScript를 활용하여 가장 능숙한 개발이 가능한 라이브러리, 다양한 패키지와 오픈소스들이 많아 개발에 다양한 이점을 가져다 줄 것이라 생각하였습니다. |
  | Recoil | 기존의 Redux나 Reducer과 비교하여 어려움이 적고 구조가 간단한 상태관리 라이브러리이기에 사용하게 되었습니다. |
  | Antd | 본 서비스에서는 대부분 직접 CSS와 HTML 작업을 하였고 아이콘에 한정하여 import 해오는 형태로 사용하였습니다. |

`백엔드`

> Typescript, Node.js, Express, mySQL, prismaORM

`인공지능`

> Python, NLTK, Scikit-learn, Pandas, Numpy, Colab & Jupyter notebook
